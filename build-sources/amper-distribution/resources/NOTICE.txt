BusyBox for Windows (busybox-w32)
===============================================================================

This product includes BusyBox for Windows ("busybox-w32"), a WIN32 native port
of BusyBox.

Project homepage:
    https://frippery.org/busybox/

Source repositories:
    https://github.com/rmyorston/busybox-w32

Upstream version:

        licenses/GPL-2.txt

Included binaries:
    - bin/busybox64u.exe   (Windows x86_64 / AMD64 build)
    - bin/busybox64a.exe   (Windows ARM64 build)

Provenance of the binaries:
    The above binaries are unmodified, prebuilt executables obtained directly
    from the official busybox-w32 distribution site:
        https://frippery.org/busybox/
    They were produced by the upstream busybox-w32 maintainer. This project
    does not compile, patch, or otherwise modify them; it redistributes them
    as-is.

Modifications:
    No modifications have been made to the upstream binaries or to the
    upstream source code by this project.

Corresponding source code (GPLv2 §3(a)):
    Pursuant to GNU GPL version 2, section 3(a), the complete corresponding
    source code for the above binaries is included in this distribution at:
        third-party-sources/busybox-w32-@BUSYBOX_W32_VERSION@.tgz

    Additionally, the source code along with the git tag can be found at:
        https://github.com/rmyorston/busybox-w32/releases/tag/@BUSYBOX_W32_VERSION@

    This tarball is the upstream busybox-w32 source tree at the revision
    that corresponds to the included binaries. It contains the full source
    code together with the scripts used to control compilation, namely the
    in-tree Makefile and Kconfig build system. The Windows build
    configurations are provided in-tree as the standard mingw64 defconfigs
    shipped by upstream:
        - configs/mingw64u_defconfig
        - configs/mingw64a_defconfig
